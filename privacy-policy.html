Universal Summarizer - Privacy Policy

Last updated: April 18, 2026

What This Extension Does

Universal Summarizer extracts text content from the webpage you are viewing and sends it to the Anthropic API to generate a structured summary. The extension only activates when you click the extension icon or press the keyboard shortcut.

Data Collection

This extension does not collect, transmit, or sell any personal data. Specifically:

No analytics or tracking. We do not use any analytics services, cookies, or tracking pixels.

No data sent to our servers. The extension has no backend server. All processing happens locally in your browser or via the Anthropic API.

No browsing history collected. The extension does not monitor or record your browsing activity.

Data That Leaves Your Browser

When you click "Summarize," the text content of the current webpage is sent to the Anthropic API (api.anthropic.com) along with your API key for processing. This is the only external communication the extension makes. Anthropic's privacy policy governs how they handle this data: anthropic.com/privacy.

Data Stored Locally

The following data is stored in your browser's local extension storage (chrome.storage.local) and never leaves your device:

Your API key — stored locally so you don't have to re-enter it each time.

Cached summaries — recent summaries are cached by URL for 24 hours to avoid redundant API calls.

Summary history — your last 50 summaries are stored locally for your reference.

You can clear all locally stored data at any time by removing the extension from your browser.

Permissions Explained

activeTab — allows the extension to read the content of the tab you are currently viewing, only when you activate the extension.

storage — allows the extension to save your API key, cached summaries, and history locally.

scripting — allows the extension to inject a content extraction script into the active tab to read page text.

sidePanel — allows the extension to display the summary in a browser side panel.

host_permissions (api.anthropic.com) — allows the extension to send requests to the Anthropic API for summarization.

Third-Party Services

The only third-party service used is the Anthropic API. No other external services are contacted.

Children's Privacy

This extension is not directed at children under 13. We do not knowingly collect data from children.
